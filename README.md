# 💻 Trabalho Individual de Front-End

Exercícios da disciplina de **Front-End Essencial** na Residência em TIC/Software do Serratec.

---

## 📄 Exercício 1 – Página de Apresentação Pessoal

Página com apresentação pessoal contendo nome, parágrafo sobre você, lista de hobbies, imagem e link para rede social. O CSS deve incluir cor de fundo, título colorido, imagem com largura definida e espaçamentos com `padding`/`margin`.

---

## 🛍️ Exercício 2 – Card de Produto

Card de produto para loja virtual com imagem, nome em destaque, descrição curta, preço e botão "Comprar".

---

## 📐 Exercício 3 – Layout com Flexbox

Reprodução de um layout usando **Flexbox**, com header, sidebar, área de conteúdo principal e footer.

---

## 🛠️ Tecnologias Utilizadas

- HTML
- CSS (Flexbox)
